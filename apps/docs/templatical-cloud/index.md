# Templatical Cloud

The open-source editor gives you everything you need to build and render email templates. **Templatical Cloud** adds the features teams need when shipping email tooling at scale.

## What You Get

### AI Assistant

Generate email content from prompts, rewrite existing text for tone and clarity, and get AI-powered design suggestions. The AI understands email best practices and your template structure.

### Real-Time Collaboration

Multiple team members can edit the same template simultaneously with live cursor tracking, block-level locking, and automatic conflict resolution.

### Comments and Reviews

Add inline comments on specific blocks for review workflows. Resolve threads, mention team members, and track feedback without leaving the editor.

### Media Library

Upload, organize, and manage images with folder structure, search, and automatic image optimization. Drag images directly from the library into your templates.

### Template Scoring

Automated quality checks for email deliverability, accessibility, and best practices. Get actionable scores and suggestions before you send.

### Saved Modules

Save reusable template sections (headers, footers, product grids) and insert them across templates. Modules stay in sync across your team.

### Test Emails

Send test emails directly from the editor to verify rendering across email clients.

### Snapshots

Automatic version history with restore. Compare snapshots side-by-side and roll back to any previous version.

### MCP Integration

Connect AI agents and automation tools to the editor via the Model Context Protocol. Build, modify, and manage templates programmatically through AI.

### Multi-Tenant Architecture

Project and tenant isolation with API keys. Give each customer their own set of templates, media, and settings while managing everything from one dashboard.

### Headless API

Full programmatic access to templates, blocks, media, and rendering. Build custom workflows, batch operations, and integrations without the visual editor.
