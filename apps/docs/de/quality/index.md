# Qualität

`@templatical/quality` ist das Dachpaket für die Template-Qualitäts-Werkzeuge von Templatical.

## Verfügbar

- **[Barrierefreiheits-Linter](./accessibility/)** – deterministische Regeln, die häufige Autorenfehler erkennen (fehlender Alt-Text, niedriger Kontrast, vage CTAs, übersprungene Überschriftenebenen, …) – sowohl im Editor als auch bei Headless- / CI-Prüfungen. MIT-lizenziert, ausschließlich JSON, ESM.
