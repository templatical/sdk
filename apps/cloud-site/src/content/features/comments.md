## What it does

Review feedback anchored to blocks — not floating in a separate doc.

- **Thread per block** — Comments attach to the exact block they’re about. Move the block, the thread follows.
- **Mention, resolve, reply** — Standard review workflow with unread indicators per thread.
- **Notifications** — Pushed through the same WebSocket as collaboration. Respond while you’re editing.

## Good fits

- Cross-functional review (marketing, legal, design) without a second tool
- Client sign-off loops for agencies
- Async review across time zones
