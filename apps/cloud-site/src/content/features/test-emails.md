## What it does

Send yourself (or a reviewer) a real test email without leaving the editor.

- **One-click send** — Pick recipients, hit send. Test renders the exact template with MJML → HTML compilation.
- **Merge-tag preview** — Populate sample data for every merge tag so the email looks real, not like `{{first_name}}`.
- **Quota management** — Per-plan daily limits surfaced in the UI. No surprise cutoffs.

## Good fits

- QA cycles before production sends
- Sharing a preview with a stakeholder who hates staging URLs
- Testing rendering in the actual inbox environment you care about
